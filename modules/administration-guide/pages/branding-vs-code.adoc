= Branding Visual Studio Code - Open Source

:context: branding-vs-code

There is a functionality to apply branding to some UI elements. It is possible to customize the following elements:

* Browser tab:
** Favicon - browser tab icon
** Product name - browser tab title

* *Get Started* page:
** Product name
** Product logo - page tab icon

* *About* dialog:
** Product name

* *Product status bar* item:
** Label
** Tooltip
** Icon
** Background color

* *Product status bar* commands: 
** The names of commands for the *Product status bar* item

* The main area:
** Icon for the main area when all editors are closed

To start using the customized Visual Studio Code:

. Provide branding resources in the `branding` folder.
. Run the `branding.sh` script.
. Build a container image with the customized Visual Studio Code.
. Define an editor `.che/che-editor.yaml` that uses the custom image.
. Create a workspace from a Git repository using the custom editor.
